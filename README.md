# シフト管理ツール

## 概要

どこからでも誰でもシフトの希望提出や、シフト表の作成ができるWebアプリケーションです。
店舗ごとに独立したシフト管理環境を提供し、管理者とスタッフ間のシフト調整を効率化します。

## 主な機能

-   **認証機能:**
    -   店舗コードとパスワードによるログイン。
    -   管理者とスタッフの権限分離。
-   **スタッフ管理 (管理者向け):**
    -   スタッフの登録、編集、削除。
    -   スタッフ種別（社員、アルバイトなど）の設定。
-   **シフト希望入力:**
    -   スタッフはカレンダー形式で直感的にシフト希望を提出できます。
    -   固定の時間帯選択と、自由記述の両方に対応しています。
-   **シフト自動生成 (管理者向け):**
    -   提出された希望と、設定された必要人数に基づき、最適化されたシフト表を自動で生成します。
    -   生成されたシフトは、手動で微調整することも可能です。
-   **詳細設定 (管理者向け):**
    -   時間帯リストのカスタマイズ。
    -   曜日ごと、または平日・週末ごとで必要人数を柔軟に設定できます。
-   **データ管理:**
    -   シフト表のPDFエクスポート機能。
    -   店舗データのインポート・エクスポート機能。

## 技術スタック

-   **バックエンド:** Python, Flask
-   **フロントエンド:** JavaScript, HTML, CSS
-   **データ永続化:** ファイルシステム (JSON)
-   **外部アクセス用ツール:**
    -   localhost.run
    -   ngrok (pyngrok)
    -   Cloudflare Tunnel

## セットアップと実行方法

### 本番環境 (Render)

RenderでホスティングされているURLにアクセスし、店舗コードとパスワードでログインします。
新しい店舗コードでログインすると、自動的に新規店舗として登録されます。

### ローカル開発環境

1.  **リポジトリをクローンします。**
2.  **Pythonの仮想環境を作成し、有効化します。**
    ```bash
    python -m venv .venv
    # Windows
    .venv\Scripts\activate
    # macOS / Linux
    source .venv/bin/activate
    ```
3.  **必要なライブラリをインストールします。**
    ```bash
    pip install Flask Flask-Session pyngrok requests reportlab
    ```
4.  **Flaskアプリケーションを起動します。**
    ```bash
    python app.py
    ```
5.  ブラウザで `http://127.0.0.1:5000` にアクセスします。

### ローカルサーバーの外部公開

スマートフォンなど、他のデバイスからアクセスする場合は、以下のいずれかのスクリプトを実行します。

-   **localhost.run を使用する場合 (認証不要):**
    ```bash
    python start_localhostrun.py
    ```

表示されたURLにアクセスすることで、外部からローカルサーバーに接続できます。
